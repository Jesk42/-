# Jesk42

Премиум-сайт: чёрно-белый минимализм, glassmorphism, неоновое свечение, частицы и дым на фоне.

## Структура

```
index.html
style.css
script.js
music/   — mp3-треки плеера
images/  — фотография профиля
```

## Как добавить/убрать трек

GitHub Pages — статичный хостинг, он не умеет самостоятельно "сканировать" папку `music` в браузере.
Поэтому список треков задаётся один раз в `script.js`, в самом верху файла, массивом `PLAYLIST`:

```js
const PLAYLIST = [
  { file: "Название файла.mp3", title: "Название трека", artist: "Исполнитель" },
  // добавьте сюда новую строку для нового трека
];
```

1. Положите mp3-файл в папку `music/`.
2. Добавьте строку в `PLAYLIST` с точным именем файла.
3. Сохраните — трек появится в плеере автоматически (карточка, длительность, воспроизведение).

## Публикация на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него все файлы и папки из этого архива (сохраняя структуру).
3. В репозитории откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Source: Deploy from a branch**, ветку `main` и папку `/ (root)`.
5. Сохраните — через минуту сайт будет доступен по адресу вида
   `https://<ваш-логин>.github.io/<название-репозитория>/`.

## Технологии

Чистый HTML / CSS / JavaScript, без фреймворков. Шрифты подключены с Google Fonts (Unbounded, Manrope, Space Mono).
