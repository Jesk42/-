# Как запустить бота, который управляет сайтом

## Шаг 1. Создай бота в Telegram
1. Открой Telegram, найди **@BotFather**.
2. Напиши ему `/newbot`.
3. Придумай имя боту и его логин (должен заканчиваться на `bot`).
4. BotFather пришлёт тебе **токен** — длинную строку вида
   `123456789:AAExxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`. Сохрани его, он понадобится дальше.

## Шаг 2. Запусти сервер на Replit (бесплатно, без установки)
1. Зайди на **replit.com** и зарегистрируйся.
2. Нажми **Create Repl** → выбери шаблон **Node.js**.
3. Слева в файлах удали содержимое `index.js` (или как он называется) и вместо него
   создай два файла: `server.js` и `package.json` — скопируй в них содержимое
   из файлов, которые я тебе дал.
4. В файле `server.js` найди строку:
   ```
   const BOT_TOKEN = 'ВСТАВЬ_СЮДА_СВОЙ_ТОКЕН';
   ```
   и вставь туда токен из Шага 1 (в кавычках).
5. Нажми зелёную кнопку **Run**. Replit сам поставит нужные пакеты и запустит сервер.
6. Справа появится окошко с адресом сайта вида
   `https://имя-репла.твой-ник.repl.co` — это и есть твоя **ссылка сервера**.
   Скопируй её.

## Шаг 3. Подключи сайт к боту
1. Открой свой сайт (`index.html`).
2. Перейди на вкладку **«Бот»**.
3. Вставь скопированную ссылку в поле и нажми **«Подключиться»**.
4. Статус должен стать «подключено ✅».

## Шаг 4. Пиши боту в Telegram!
Открой своего бота в Telegram (по ссылке t.me/твой_бот) и напиши, например:
- `снег` — снежинки на сайте включатся/выключатся
- `музыка` — включит/выключит музыку
- `чёрный` — выключит "экран" на сайте
- `трясти` — потрясёт страницу
- `переверни` — перевернёт страницу
- `цвет` — инвертирует цвета

Через 1–2 секунды сайт должен отреагировать!

## Важно
- Пока Replit открыт и запущен — бот работает. Если закрыть вкладку Replit,
  бот "уснёт", и команды перестанут доходить. Бесплатный Replit не работает 24/7 —
  это нормально для пробы.
- Хочешь добавить свою команду? Открой `server.js`, найди список `COMMANDS`
  и добавь новую строку по образцу, например:
  ```
  'привет': 'hello',
  'зелёный': 'green',
  ```
  Потом в `index.html` в функции `applyBotCommand` добавь, что должно происходить
  для этой команды.
